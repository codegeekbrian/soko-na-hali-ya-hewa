A simple web platform for Kenyan small-scale farmers: Real-time crop prices, weather alerts, and buyer connections. Addresses food insecurity and low incomes in Rift Valley/Western Kenya.

## Features
- **Crop Prices**: Select market (Eldoret, Kisumu, Nakuru) for maize/beans/potatoes (KSh/kg).
- **Weather Alerts**: Enter location for forecasts via OpenWeatherMap API.
- **Buyers Directory**: Contacts for co-ops and traders.
- Mobile-first, no login needed.

## Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Free APIs: OpenWeatherMap
- Hosting: GitHub Pages

## Setup
1. Clone: `git clone https://github.com/codegeekbrian/soko-na-hali-ya-hewa.git`
